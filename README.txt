NUGZ v2 — iSmokeShop
=====================
Files required in root folder:
  index.html
  styles.css
  index.js
  nugz-background.png
  logo.png
  sprites/  (folder with all .png files)

MUSIC: Place "NUGZ Main Menu (Take 1).mp3" in the root folder.
The game auto-detects it. Music is OFF by default — player
enables it in OPTIONS then it plays on menu + game screens.

Deploy to GitHub Pages or any static web server.
Open index.html locally or via http:// — NOT file:// for music.
